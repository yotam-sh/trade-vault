הצהרת נגישות — TradeVault

שם האתר/האפליקציה: TradeVault — מעקב תיק השקעות

תאריך הצהרה זו: מרץ 2026

רמת נגישות

אפליקציה זו שואפת לעמוד בדרישות תקן ישראלי IS 5568, המבוסס על הנחיות WCAG 2.1 ברמה AA (רמה בינונית-גבוהה).

אמצעי נגישות שיושמו

  • קישור "דלג לתוכן הראשי" בראש כל עמוד, גלוי בעת ניווט במקלדת
  • ניווט מלא במקלדת — כל הפעולות האינטראקטיביות נגישות ללא עכבר
  • סמן מיקוד גלוי בכל אלמנט אינטראקטיבי
  • תפריט ניווט ראשי עם ARIA landmark ותוויות מתאימות
  • הגדרת שפה וכיוון (RTL/LTR) בתג <html>
  • כפתורי מיון בטבלאות עם תוויות aria-sort
  • לחצן לוח שנה עם aria-expanded וניווט חצים במקלדת
  • הכרזת תאריך נבחר לקוראי מסך (aria-live="polite")
  • גרפים וtreemap עם תוויות alt וניווט מקלדת לתאים
  • הודעות Flash מוכרזות מיידית לקוראי מסך (role="alert")
  • אזור תוכן ראשי מסומן (role="main")
  • הדף הפעיל בניווט מסומן עם aria-current="page"

מגבלות ידועות

  • גרפי Chart.js (תרשים ערך תיק, תשואה חודשית) הם אלמנטים ויזואליים; הנתונים הגולמיים זמינים בטבלאות בעמודים האחרים
  • מפת תיק (Treemap) — הנתונים זמינים גם בעמוד הפוזיציות בפורמט טבלה

משוב ויצירת קשר

אם נתקלת בבעיית נגישות באפליקציה זו, אנא פנה אלינו כדי שנוכל לסייע ולשפר את הנגישות.

ניתן לשלוח משוב בדף הגיטהאב של הפרויקט: github.com/yotam-sh/trade-vault

מידע נוסף

לפרטים על תקן IS 5568 ועל דרישות נגישות אתרי אינטרנט בישראל, ניתן לפנות ל נציבות שוויון זכויות לאנשים עם מוגבלות .

Accessibility Statement — TradeVault

Application name: TradeVault — Portfolio Tracker

Statement date: March 2026

Conformance Level

This application aims to conform to Israeli Standard IS 5568, which is based on WCAG 2.1 Level AA (Web Content Accessibility Guidelines).

Accessibility Features Implemented

  • Skip-to-main-content link at the top of every page, visible on keyboard focus
  • Full keyboard navigation — all interactive actions are accessible without a mouse
  • Visible focus indicator on all interactive elements
  • Main navigation with ARIA landmarks and appropriate labels
  • Language and direction (RTL/LTR) declared on the <html> element
  • Table sort buttons with aria-sort state announcements
  • Calendar picker button with aria-expanded and arrow-key navigation
  • Selected date announced to screen readers via aria-live="polite"
  • Charts and treemap cells have alt labels and keyboard navigation
  • Flash messages announced immediately to screen readers via role="alert"
  • Main content area marked with role="main"
  • Active navigation link marked with aria-current="page"

Known Limitations

  • Chart.js charts (portfolio value, monthly return) are visual elements; the underlying data is available in tabular form on other pages
  • Portfolio treemap — data is also available in the Positions page as a table

Feedback and Contact

If you encounter an accessibility barrier in this application, please contact us so we can assist and improve accessibility.

Feedback can be submitted on the project's GitHub page: github.com/yotam-sh/trade-vault

Further Information

For details about IS 5568 and web accessibility requirements in Israel, see the Commission for Equal Rights of Persons with Disabilities .
